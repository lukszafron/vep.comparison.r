# vep.comparison.r

Error: This program requires eight arguments. The first one is the destination folder, the second is the number of threads to use, the third should point to the CSV file used in the vep.r analysis, the fourth is the grouping variable, the fifth is the independent factor used, the sixth determines if the samples are paired, the seventh indicates the pair indicator column, while the eighth determines whether the FDR correction should be performed.
